Bookshelf API

This API was created to organize book management in a library such as information about books, managing readings and others. This API provides CRUD operations for books, including adding new books, viewing the book list, updating book information, and deleting books from the list.


This Bookshelf API is built using Node.js with the ExpressJS framework, HapiJS for the server side. For the database, it uses MySQL to store book information with Sequelize ORM.

JWT Auth API

This API has the following main features: Allows users to register and create new accounts with valid information, Generates a JWT token after a user successfully logs in with the correct credentials, Uses the JWT token to secure certain routes, ensuring only authenticated users can access them.


This JWT API is built using Node.js with the ExpressJS framework for the server side. For the database, it uses MySQL with Sequelize ORM.

ZFILMS Project
